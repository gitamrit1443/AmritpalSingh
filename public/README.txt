This directory is served as the root of the built app.
Place your favicon.ico, robots.txt, and any other root-level static files here.

Recommended files to add:
  - favicon.ico          (your personal favicon)
  - robots.txt           (see below)
  - sitemap.xml          (optional, for SEO)
  - og-image.png         (1200×630px Open Graph preview image)

--- robots.txt sample ---
User-agent: *
Allow: /
Sitemap: https://yourdomain.com/sitemap.xml
